# Deploy to Cloudflare Pages

تم إنشاء ملف wrangler.toml لنشر الموقع على Cloudflare Pages

## الخطوات:

1. اذهب إلى: https://pages.cloudflare.com
2. اختر "Connect to Git"
3. اختر المستودع: ahaboelsoud10-hash/-ahmed-ai-
4. اختر الفرع: main
5. اضغط Deploy

الموقع سيكون متاح على رابط مثل:
https://ahmed-ai-xxxx.pages.dev
